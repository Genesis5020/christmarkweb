Frequently Asked Questions

What does a SOC Analyst do?
Monitors networks and systems for security threats, analyzes logs, responds to incidents, and helps improve security defenses.
What tools and technologies do you use?
SIEM tools, firewalls, IDS/IPS, Wireshark, Nmap, Python, Linux, Windows Security, and various threat intelligence platforms.
How do you handle security incidents?
Follow standard incident response process: Identify → Contain → Eradicate → Recover → Lessons Learned.
Can you help with network security setup?
Yes — I can help configure security devices, policies, and best practices to protect your network.
